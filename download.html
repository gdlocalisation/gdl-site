Скачать установщик для : Скачать

Будет скачан установщик с расширением .exe, который автоматически выберет лучший способ установки. Варианты:

  • MegaHack v6.2+ - мод (.dll) будет помещён в папку "extensions".

  • ModLdr от HJfod - мод (.dll) будет помещён в папку "mods".

  • ProxyDLLLoader от Adaf - файл будет помещён в папку "adaf-dll".

  • Если не будет найден ни один загрузчик модов, то файл мода (.dll) будет помещён в папку с GeometryDash.exe. Также будет модифицирован файл libExtensions.dll, чтобы автоматически загружать мод при запуске игры.


Также будет скачан файл "ru_ru.json", содержащий перевод, и будут изменены некоторые текстуры.

При необходимости удалить русификатор вы можете либо выбрать запустить gdl_unins000.exe, либо самостоятельно восстановить игровые файлы (версия Steam).
